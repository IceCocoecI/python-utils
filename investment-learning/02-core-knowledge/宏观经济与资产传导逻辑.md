## 为什么要懂宏观？

你买的每一只股票、基金，都漂浮在宏观经济这片"海"上。**公司基本面决定船好不好，宏观决定潮水涨不涨。**

- 美联储加息 → 全球风险资产承压，你的 A 股/港股/美股可能一起跌。
- 美元走强 → 资金从新兴市场回流美国，港股被抽血。
- 原油暴涨 → 通胀预期抬头 → 利率预期上升 → 成长股杀估值。

不懂宏观，你会发现"公司明明没变差，股价却一直跌"——因为潮水退了。

## 第一部分：核心宏观变量（逐个讲清）

### 1. 美联储（Fed）：全球资产定价的"总开关"

#### 是什么

美国的中央银行，全称"联邦储备系统"。因为美元是全球储备货币，美联储的政策影响力**远超美国国境**，是全球资产定价的核心变量。

#### 双重使命（Dual Mandate）

美联储的法定目标只有两个：

1. **物价稳定**（控制通胀，目标 2%）
2. **充分就业**（失业率维持在自然失业率附近）

美联储的所有操作——加息、降息、缩表、扩表——都是围绕这两个目标做平衡。

#### 联邦基金利率（Fed Funds Rate）

美联储的核心政策工具。它是银行之间**隔夜拆借**的利率，虽然你不直接接触，但它是整个利率体系的"锚点"——所有其他利率（房贷、企业贷、国债）都围绕它上下浮动。

---

### 2. 加息与降息：最核心的政策动作

#### 加息（收紧 / Tightening）

**什么时候加息**：经济过热、通胀太高时。

**传导链条**：

```text
美联储加息
  → 银行间利率上升
    → 企业贷款成本上升 → 投资减少 → 经济降温
    → 消费者房贷/信用卡利率上升 → 消费减少 → 需求降温
    → 储蓄回报提高 → 人们更愿意存钱而非消费/投资
      → 需求下降 → 通胀被压制（目标达成）
```

**对资产的影响**：

- **股市**：整体承压。折现率上升 → 估值下降（尤其**远期盈利支撑的成长股/科技股**受打击最大）。
- **债市**：利率上升 → 已有债券价格下跌（利率与债券价格**反向**运动）。
- **美元**：倾向走强（高利率吸引资金流入美国）。
- **黄金**：倾向承压（持有黄金无利息，利率越高、持有黄金的机会成本越大）。

#### 降息（宽松 / Easing）

**什么时候降息**：经济衰退风险上升、就业恶化、通胀达标或低于目标时。

**传导链条**（加息反过来）：

```text
美联储降息
  → 贷款便宜 → 企业扩张、消费者敢花钱
    → 经济活动回暖
    → 储蓄回报低 → 资金被"逼"向风险资产（股票、房产等）
      → 资产价格上涨
```

**对资产的影响**：

- **股市**：利好（尤其成长股/科技股，折现率下降 → 远期估值拉高）。
- **债市**：利好（利率下降 → 债券价格上涨）。
- **美元**：倾向走弱。
- **黄金**：倾向上涨（机会成本降低 + 通常伴随不确定性）。

#### 关键提醒

- 市场是**预期机器**：真正影响价格的不是"加息/降息本身"，而是**"比市场预期多还是少"**。如果市场已经预期加息 3 次，实际只加了 2 次，反而是利好。
- **不要线性外推**："加息就一定跌"是过度简化。有时"靴子落地"反而涨（利空出尽）。

### 3. 通货膨胀（Inflation）：为什么这么重要

#### 是什么

物价持续上涨的过程。你的钱买到的东西越来越少。

#### 核心指标

- **CPI（消费者价格指数）**：衡量一篮子消费品价格变动，最常用的通胀指标。
- **核心 CPI**：剔除波动大的食品和能源，反映"底层通胀趋势"，**美联储更看重这个**。
- **PCE（个人消费支出物价指数）**：美联储最青睐的通胀度量。目标是 **PCE 年化 2%**。
- **PPI（生产者价格指数）**：上游出厂价格，是 CPI 的"先行指标"——PPI 涨，迟早传导到消费端。

#### 通胀的两种成因

1. **需求拉动型**：钱太多、需求太旺（如 2020-2021 年美国大规模发钱后消费暴增）。
2. **成本推动型**：供给端出问题（如原油暴涨推高运输和生产成本、供应链断裂）。

#### 通胀对投资的影响

- **温和通胀（2% 左右）**：健康，企业能提价、盈利增长，股市友好。
- **高通胀（远超 2%）**：迫使央行加息 → 前面讲的所有紧缩传导链启动 → 股债双杀。
- **通缩（物价下跌）**：听起来好像"东西便宜了"，实际上非常危险——消费者推迟消费（"明天更便宜"）→ 企业收入萎缩 → 裁员 → 螺旋式衰退。

---

### 4. 就业数据：经济的体温计

#### 核心指标

- **非农就业数据（Non-Farm Payrolls, NFP）**：每月第一个周五发布，全球金融市场**最敏感的数据之一**。反映当月新增了多少就业岗位（不含农业）。
- **失业率**：总失业人口占劳动力的比例。
- **时薪增速**：工资涨得快不快，直接关系消费力和通胀。

#### 怎么解读

- **非农大超预期（就业强劲）** → 经济强 → 但也意味着通胀可能更顽固 → **市场担心美联储不敢降息/继续加息** → 短期可能利空股市。
- **非农大幅低于预期（就业冷却）** → 经济弱 → 美联储可能加快降息步伐 → **短期可能利好股市**（"坏消息变好消息"逻辑）。
- **时薪增速过高** → 工资-物价螺旋风险（涨工资 → 企业提价 → 更高通胀 → 再涨工资…）→ 美联储更鹰派。

#### "坏消息就是好消息"的怪象

在加息周期尾声，市场**最盼望**的反而是经济数据变差——因为只有经济冷下来，美联储才会停止加息甚至降息。这时弱就业=利好，强就业=利空。**反直觉，但非常常见。**

### 5. 扩表与缩表（QE / QT）：印钱与收钱

#### 量化宽松 QE（扩表 / 印钱）

**什么时候用**：利率已经降到接近 0 还不够，需要"非常规手段"进一步宽松时（如 2008 金融危机、2020 疫情）。

**怎么做**：美联储在市场上**大量买入国债和 MBS（房贷支持证券）**，把钱注入金融体系。

```text
美联储买债 → 向市场注入大量现金（流动性）
  → 国债价格上升 → 国债利率下降（价格与利率反向）
    → 整体利率水平被压到极低
    → 企业和消费者借钱更容易
    → 大量资金无处安放，涌入股市、房市等风险资产
      → 资产价格暴涨（2020-2021 全球大牛市的主因）
```

美联储的资产负债表因此**膨胀**（买了一堆债券），所以叫"扩表"。

#### 量化紧缩 QT（缩表 / 收钱）

**什么时候用**：宽松过度、通胀飙升后，需要把之前放出的水"收回来"。

**怎么做**：美联储让到期的国债**不再续购**（到期不投 → 钱回到美联储 → 从市场消失），或直接卖出持有的债券。

```text
美联储缩表 → 从市场回收流动性
  → 市场上的"钱"变少
    → 利率抬升（资金供给减少，借钱更贵）
    → 风险资产失去"水"的支撑
      → 股市、债市承压
```

#### QE/QT 的本质

- **QE = 央行给市场"注水"**，水位上升，所有船（资产价格）被抬高。
- **QT = 央行从市场"抽水"**，水位下降，船搁浅。
- 最重要的不是绝对水量，而是**水量变化的方向和速度**——从"放水"切换到"抽水"的拐点，往往是市场最剧烈波动的时候。

### 6. 美国国债收益率（美债利率）

#### 是什么

你借钱给美国政府能拿到的年化利率。因为美国国债被视为全球最安全的资产，美债收益率是**全球无风险利率的基准**。

#### 关键期限

- **2 年期**：反映市场对**近期**（1-2 年内）美联储加息/降息的预期，和联邦基金利率走势高度相关。
- **10 年期**：反映市场对**长期经济增长和通胀**的预期，是房贷利率、企业债利率的定价基础——**最重要的单一利率指标**。
- **30 年期**：更长期的预期。

#### 收益率曲线与倒挂

正常情况：期限越长利率越高（锁定更久，要求更多补偿）→ 收益率曲线**向上倾斜**。

**收益率曲线倒挂**：短期利率（如 2 年期）反超长期利率（如 10 年期）。

这意味着市场认为"短期经济还行但长期要出事"→ 预期未来会大幅降息。**历史上几乎每次美债 2/10 年倒挂后，1-2 年内都出现了经济衰退**——被视为最可靠的衰退预警信号之一。

#### 对资产的影响

```text
10 年期美债收益率上升
  → 股市折现率上升 → 估值承压（尤其成长股）
  → 新发债券收益更高 → 老债券价格下跌
  → 美元倾向走强
  → 新兴市场（含 A 股/港股）资金外流压力增大
```

### 7. 美元强弱

#### 衡量方式

- **美元指数（DXY）**：美元对一篮子主要货币（欧元占比最大）的综合强弱。DXY 上涨 = 美元走强。

#### 什么驱动美元强弱

```text
美元走强的典型场景：
  - 美联储加息/鹰派 → 美元资产利率更高 → 全球资金流入美国
  - 美国经济强于其他国家 → 投资回报预期高
  - 全球避险情绪升温（危机时资金涌入美元/美债避险）

美元走弱的典型场景：
  - 美联储降息/鸽派 → 美元资产吸引力下降
  - 美国经济放缓而其他地区复苏 → 资金外流
  - 全球风险偏好改善 → 资金流向高收益的新兴市场
```

#### 美元强弱对各类资产的影响

| 资产 | 美元走强 | 美元走弱 |
| --- | --- | --- |
| 美股 | 双面（跨国企业海外收入缩水，但资金流入支撑估值） | 跨国企业收入改善 |
| 新兴市场（A股/港股） | **利空**（资金回流美国、本币贬值、美元债务成本上升） | **利好**（资金流入、本币升值） |
| 大宗商品（以美元计价） | **承压**（同样的商品需要更少美元） | **利好**（美元贬值推高商品价格） |
| 黄金 | 承压 | 利好 |
| 你持有的美元资产 | 换回人民币时**赚汇率** | 换回人民币时**亏汇率** |

#### 关键记忆

**美元走强 = 全球流动性收紧**（美元是全球储备货币，美元升值意味着全世界的"钱"相对变少了）。这就是为什么美元走势对**非美市场**影响巨大。

### 8. 黄金：避险 + 抗通胀 + 反美元

#### 黄金的三重属性

1. **避险资产**：地缘冲突、金融危机、经济不确定性升温时，资金涌入黄金避险。
2. **抗通胀**：法定货币贬值（超发），黄金作为"不能被印出来的货币"保值。
3. **与美元/实际利率反向**：黄金以美元计价，美元弱则金价强；黄金不产生利息，**实际利率**（名义利率 − 通胀）越低，持有黄金的机会成本越低 → 金价越有支撑。

#### 核心定价逻辑

```text
实际利率 = 名义利率（美债利率）− 通胀预期

实际利率下降 → 黄金上涨（最核心的驱动）
实际利率上升 → 黄金承压
```

**直观理解**：如果国债利率 5% 但通胀 6%，实际利率 = −1%，存钱等于亏钱 → 资金涌入黄金。如果国债利率 5% 通胀只有 2%，实际利率 = 3%，持有国债就很舒服 → 黄金吸引力下降。

#### 黄金上涨的典型环境

- 降息周期 + 通胀仍高（实际利率走低）
- 地缘危机 / 金融系统不稳定
- 美元走弱
- 全球央行增持黄金储备（近年来趋势明显）

#### 常见误区

- "通胀高黄金就涨"——不一定。如果为了压通胀大幅加息，名义利率涨得比通胀还快，实际利率反而上升，黄金可能跌（如 2022 年上半年）。**关键看实际利率，不是通胀本身。**
- 黄金短期波动大，**不适合"追涨"**，更适合作为组合中的长期配置（5%~15%）。

### 9. 原油：全球经济的血液

#### 为什么重要

原油是工业生产、运输、化工的基础原料。油价变动会**传导到几乎所有商品和服务的成本**。

#### 油价由什么决定

**供给侧**：

- **OPEC+**（石油输出国组织及盟友，尤其沙特/俄罗斯）：通过**减产/增产**控制供给，是油价最大的人为调控变量。
- 地缘冲突（中东局势、俄乌战争等）→ 供给中断恐慌 → 油价飙升。
- 美国页岩油产量变化。

**需求侧**：

- 全球经济增长 → 工业/运输需求旺 → 油价涨。
- 经济衰退 → 需求萎缩 → 油价跌。
- **中国经济**是全球原油需求的重要边际变量。

#### 油价对通胀和利率的传导

```text
油价暴涨
  → 运输/生产成本上升
    → PPI 上涨 → 传导至 CPI → 通胀抬头
      → 美联储可能被迫加息/维持高利率
        → 股市承压

油价暴跌
  → 成本下降 → 通胀缓解
    → 美联储有降息空间
      → 股市利好
  → 但如果是因为经济衰退导致的需求崩塌 → 就不是好事（衰退信号）
```

#### 油价与各类资产

- **通胀预期**：油价是通胀的"先行指标"之一。油价飙升 → 市场立刻上调通胀预期。
- **能源股**：油价涨 → 能源公司（中石油、壳牌、埃克森美孚）利润暴增。
- **航空/运输/化工**：油价涨 → 成本飙升，利润被挤压。
- **消费**：油价涨 → 加油/取暖贵 → 消费者可支配收入减少 → 消费股承压。

#### 常见误区

- 油价下跌不一定是好事——要区分是"供给增加"（好事，成本降低）还是"需求崩塌"（坏事，经济衰退）。

## 第二部分：传导全景图（把所有变量串起来）

宏观分析最难的不是理解单个变量，而是理解它们之间的相互影响和连锁反应。

### 典型传导链条 1：通胀螺旋

```text
油价暴涨 / 供应链断裂
  → 生产成本上升（PPI↑）
    → 企业提价 → 消费价格上涨（CPI↑）
      → 工人要求涨工资
        → 企业成本再上升 → 再提价（工资-物价螺旋）
          → 美联储被迫大幅加息
            → 股市暴跌、债市暴跌、经济降温
              → 最终通胀被压下来，但代价是衰退
```

**真实案例**：2021-2022 年，疫情后供应链紊乱 + 能源危机 + 前期大放水 → 美国 CPI 飙至 9% → 美联储史上最快加息周期 → 2022 全球股债双杀。

### 典型传导链条 2：衰退与宽松

```text
经济增速放缓
  → 企业盈利下滑 + 裁员
    → 非农数据恶化、失业率上升
      → 消费进一步萎缩
        → 通胀自然回落
          → 美联储开始降息 + 可能重启 QE
            → 利率下降 → 资金流入股市债市
              → 经济触底 → 新一轮复苏
```

**真实案例**：2020 年 3 月，疫情冲击 → 经济瞬间冻结 → 美联储紧急降息至 0 + 无限量 QE → 股市 V 型反转。

### 典型传导链条 3：美元升值冲击新兴市场

```text
美联储加息 + 美国经济相对强劲
  → 美元走强（DXY↑）
    → 资金从新兴市场撤出（套利资金回流高利率美国）
      → 新兴市场货币贬值
        → 这些国家的美元债务负担加重
        → 进口成本上升（输入型通胀）
          → 新兴市场央行被迫跟着加息（否则货币崩盘）
            → 本国经济被双重挤压（外部紧缩 + 内部被动紧缩）
              → 港股、A 股都可能被"连坐"
```

**真实案例**：2022 年，美联储激进加息 → 美元指数飙升至 114 → 日元、人民币、英镑大幅贬值 → 港股跌至多年低位。

### 典型传导链条 4：黄金的"甜蜜点"

```text
经济放缓 + 通胀仍高（"滞胀"环境）
  → 美联储左右为难（加息怕衰退，降息怕通胀）
    → 实际利率走低或为负（名义利率上不去，通胀下不来）
      → 黄金大涨
    → 同时地缘风险叠加 → 避险需求 → 黄金再叠加一层上涨动力
```

| --- | --- | --- | --- | --- | --- |

## 第三部分：几个常见困惑

### Q1："美联储鹰派 / 鸽派"是什么意思？

- **鹰派（Hawkish）**：倾向紧缩（加息、缩表），优先打压通胀。
- **鸽派（Dovish）**：倾向宽松（降息、扩表），优先保就业/经济增长。
- 每次美联储会议后的声明和主席讲话，市场会**逐字解读**是偏鹰还是偏鸽，据此调整预期。一个词的变化（如从"持续加息"变为"视数据而定"）就可能引发市场大幅波动。

### Q2：为什么有时"利率已经不涨了"股市还是跌？

因为利率是**水平值**，不是变化率。利率从 0 涨到 5% 后即使不再涨，5% 这个水平本身就是对经济和估值的持续压力。只有当利率开始**下降**，压力才真正缓解。所以"停止加息"不等于"利好"——市场盼的是**降息**。

### Q3：A 股受美联储影响大吗？

大，而且越来越大。传导路径至少有三条：

1. **利率传导**：美债利率上升 → 全球无风险利率中枢抬升 → A 股估值也受压。
2. **汇率传导**：美元走强 → 人民币贬值 → 外资撤出 A 股/港股。
3. **情绪传导**：美股暴跌 → 全球恐慌 → A 股跟跌。

但 A 股也有**自身节奏**（中国的货币政策、财政刺激、产业政策），不完全和美股同步。

### Q4：怎么跟踪这些数据？

- **美国非农/CPI/PCE**：每月发布，财经日历会标注（关注"前值 vs 预期 vs 实际"）。
- **美联储利率决议**：每年 8 次（约 6 周一次），附点阵图和经济预测。
- **美元指数 / 美债收益率 / 金价 / 油价**：实时行情，各财经 APP 均可看。
- **CME FedWatch 工具**：市场对未来加息/降息概率的实时投票，极其有用。

## 给初学者的建议

1. **不要试图预测宏观**——连美联储自己都经常预测错。你要做的是**理解传导逻辑、建立方向感**，而不是赌某个数据。
2. **资产配置比猜宏观重要**——正因为宏观不可预测，所以才要分散：股债搭配、多市场配置、保留现金。不管潮水往哪个方向涌，你的组合都不会被淹死。
3. **关注"预期差"而不是数据本身**——市场已经 price in 的信息不再是新闻。超预期的部分才会推动价格。
4. 这些传导链条**需要反复在真实市场中验证**——每次重大数据发布后，回头对照"发生了什么→市场怎么反应→符不符合传导逻辑"，慢慢就会建立直觉。
