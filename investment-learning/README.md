# 投资理财学习 (Investment Learning)

投资理财相关的学习笔记与分析模板，目前以 Markdown 文档为主。

## 模块结构

```
investment-learning/
├── 01-analysis/                  # 一、具体分析（个股、基金等的详细分析）
│   ├── 01-stocks/                # 股票
│   │   ├── PROMPT.md             #   分析提示词（板块级，一份）
│   │   ├── hk-港股/
│   │   ├── us-美股/
│   │   ├── a-share-A股/
│   │   ├── global-全球/
│   │   └── other-其他市场/
│   ├── 02-funds/                 # 基金
│   ├── 03-futures/               # 期货
│   └── 04-macro/                 # 宏观
└── 02-core-knowledge/            # 二、核心知识点（投资理财相关与核心知识）
```

## 说明

- **01-analysis（具体分析）**：每个板块（股票 / 基金 / 期货 / 宏观）各有一份分析提示词 `PROMPT.md`，
  并按市场细分为 港股 / 美股 / A股 / 全球 / 其他市场，存放个股、基金等标的的详细分析文档。
- **02-core-knowledge（核心知识点）**：投资理财相关概念与核心知识的系统性整理。
