# 05 · 横向对比 (Cross-Stock Comparison)

跨标的、跨市场的「多股横向对比」分析文档。当一个投资主题涉及多家可比公司（同赛道竞争者 / 产业链上下游 / 多市场同概念龙头）时，单票备忘录不便于直接比价，故在此集中做「同框对比 + 赔率排序」。

## 定位

- 与 `us-stocks-美股/`、`a-share-A股/`、`hk-stocks-港股/` 等**单票目录互补**：单票目录回答「这家公司值不值得买」，本目录回答「同一赛道里**优先买谁、谁的赔率更好**」。
- 每篇文档聚焦**一个主题 / 一条赛道**，横向拉通 3 家及以上可比标的。

## 命名规范

- 文件名：`YYYYMMDD-主题-标的1-标的2-…​.md`（一级标题含日期，便于按时间增量更新）。
- 增量更新时新增同主题、新日期的文档，旧版保留以便对比观点演变。
