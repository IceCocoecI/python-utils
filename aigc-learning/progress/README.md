# Learning Progress

> 用来记录长期学习过程。路线负责“学什么”，progress 负责“今天推进了什么、卡在哪里、产出了什么”。

---

## 文件

| 文件 | 用途 |
|---|---|
| [learning-log-template.md](./learning-log-template.md) | 每次学习后填写 |
| [weekly-review-template.md](./weekly-review-template.md) | 每周复盘 |
| [project-report-template.md](./project-report-template.md) | 每个 Lab 或作品完成后填写 |

---

## 建议节奏

- 每次学习结束后，用 `learning-log-template.md` 记录 5 分钟。
- 每周末用 `weekly-review-template.md` 做一次 20 分钟复盘。
- 每完成一个 [Lab](../labs/README.md)，用 `project-report-template.md` 写一页报告。

不要追求写得长，关键是留下命令、指标、问题和判断。
