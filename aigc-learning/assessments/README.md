# Assessments

> `assessments/` 用来判断“是否真的掌握”，不是新的教程。
> 做法很简单：按路线学习，阶段结束后回到这里自检。

---

## 文件说明

| 文件 | 用途 |
|---|---|
| [practice-checkpoints.md](./practice-checkpoints.md) | 实践路线阶段验收 |
| [theory-checkpoints.md](./theory-checkpoints.md) | 理论路线核心问题 |
| [interview-questions.md](./interview-questions.md) | 面试和复盘问题集 |

---

## 评分建议

每个条目用 0/1/2 分自评：

| 分数 | 含义 |
|---|---|
| 0 | 不会，或者只能照着教程复述 |
| 1 | 大概会，但需要查资料或不稳定 |
| 2 | 能独立完成，并能解释关键 tradeoff |

一个阶段建议达到 80% 以上再进入下一阶段。低于 60% 时，优先回到对应 Lab 补实践。
